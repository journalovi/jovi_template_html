Code

The template uses Prism to display code using syntax highlighting, including inline code (e.g. var x = 25;) and code blocks:

var x = 25; function(x) { return x * x; }

By default, the Prism modules for python, C-like languages (clike), lua, bash, go, markdown, julia, R, and markup languages (HTML, CSS, etc) are enabled. If you wish to use another language let us know and we can set it up.

Contributions

JoVI papers should include a CRediT contribution statement in the appendix indicating who did what work on the article. Depending on the number of authors and the categories covered, this could be a simple list; e.g.:

  • Matthew Kay: Conceptualization, Supervision, Writing - revising and editing
  • Amy Smith: Investigation, Writing - original draft, Writing - revising and editing
